* Installation of Angular

- Make sure you have nvm installed. To install -

#+begin_src bash
  curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.5/install.sh | bash
  source ~/.bashrc
  nvm -v.
#+end_src
